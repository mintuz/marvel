### Error

When you want to inform a user that their is an error within the field, you can do so by passing a prop `error` to the component, this prop can accept a React component or plain text and will be displayed to the user.

<Preview>
    <Story name="Input field error">
        <Theme>
            <InputField
                error="This is an error message passed in"
                label="My input label"
                placeholder="Placeholder"
            />
        </Theme>
    </Story>
</Preview>
