# Theming

Marvel provides a `<ThemeProvider>` component that is required to render components properly. When a `theme` object is not supplied, a default theme will be applied.

## Accessing the Theme

Their are several ways to access the theme object.

### Shorthand Theme Access

```
import styled from 'styled-components'
import { colorValue, typographyValue } from '@mintuz/marvel-theme'

const Component = styled.div`
    font-size: ${typographyValue('body')};
    background-color: ${colorValue('accent')};
`;
```

### CSS Helper Method

```
import styled from 'styled-components'
import { css } from '@mintuz/marvel-theme'

const Component = styled.div`
    ${css({
        fontSize: [3]
    })}
`;
```
